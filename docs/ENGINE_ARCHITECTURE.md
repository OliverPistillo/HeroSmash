# Hero Engine v0.7

Hero Engine è un mini motore 2D costruito solo per Hero Smash.

## Layer

- `src/engine`: rendering, input, scene, assets, audio, save, particles
- `src/game`: stato, economia, rami, carte, combat
- `src/scenes`: schermate di gioco
- `src/ui`: componenti visuali

## Perché

La v0.6 era ancora troppo monolitica. La v0.7 rende il progetto scalabile: più carte, più scene, bot league e mobile wrapper senza rifare tutto.
