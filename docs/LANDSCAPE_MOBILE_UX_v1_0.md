# Landscape Mobile UX v1.0

## Obiettivo

Portare Hero Smash da layout verticale prototype a layout landscape mobile.

## Pattern implementato

- Leaderboard laterale sinistra.
- Arena centrale.
- Card market in basso.
- Pannello ramo/stats/azioni a destra.
- Top bar con round e timer.
- Rotate overlay in portrait.

## Mobile UX

- Touch target > 48px.
- Bottoni con haptic feedback.
- Pointer drag con dx/dy.
- PWA manifest landscape/fullscreen.
- UI pensata per rapporto 16:9.

## Prossimi miglioramenti

- Safe-area notch avanzata per device specifici.
- Service worker/offline.
- Gesture card drag-to-buy.
- Long-press card detail.
- Slider velocità combat.
- Settings audio/haptic.
