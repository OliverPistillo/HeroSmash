# v1.11.1 — No Card Selling Correction

Correzione di gameplay: le carte non possono essere vendute.

## Regola

- Le carte acquistate restano nella build della run.
- La collection serve solo a consultare carte possedute, livello, rarità e rami.
- Non esistono refund da vendita.
- `UNDO` resta disponibile solo per annullare l'ultimo acquisto, se previsto dal market.

## Perché

La vendita aggiunge contabilità e rischio di errore su mobile. Per il flow attuale di Hero Smash è più pulito che ogni acquisto sia una scelta strategica stabile della run.
